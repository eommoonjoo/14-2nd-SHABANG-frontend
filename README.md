# 샤방 프로젝트

## 개요

#### 목적 : [직방](https://www.zigbang.com/) 을 클론하면서 리액트와 장고의 기본기를 향상한다

#### 일정 : 2020년 11월 30일 (월) ~ 12월 11일 (금), 12일간 진행

#### 참여자 :

- Frontend : 김동하, 장호철, 엄문주
- Backend : 김정식(PM), 이민영, 김민서

> 직방 페이지의 다양한 기능을 구현한다.

## 핵심 기능 Key Feature

#### 원룸, 아파트
- 웜룸 페이지
- 아파트 페이지
- 매물 디테일 페이지
- 지도 구현
- 필터 기능
- 검색 기능
